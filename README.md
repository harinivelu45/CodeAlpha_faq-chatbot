# 🤖 FAQ Chatbot using Python & Streamlit


## 📌 Project Description


This is a simple FAQ Chatbot Web Application developed using Python, Machine Learning (Scikit-learn), and Streamlit.  
The chatbot answers user queries based on trained FAQ data using TF-IDF vectorization and cosine similarity matching.

## ⚙️ How to Run Locally

### 1. Clone the repository
git clone https://github.com/your-username/faq-chatbot.git

### 2. Go to project folder
cd faq-chatbot

### 3. Install requirements
pip install -r requirements.txt

### 4. Run app
streamlit run app.py

---

## 💡 How It Works
- User enters a question
- TF-IDF converts text into vectors
- Cosine similarity finds best match
- Bot returns the correct answer
